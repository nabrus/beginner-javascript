Truthy Values

  • true
  • 1
  • -10
  • A string of zero ("0")
  • A full string ("Any non-empty string")
  • An empty array ([])
  • An empty object ({})

Falsy Values

  • false
  • 0
  • Empty string ("")
  • undefined variable
  • Variable set to null
  • A variable set to `"hello" - 10`. Which is NaN
